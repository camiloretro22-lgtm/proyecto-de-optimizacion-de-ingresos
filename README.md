# proyecto-de-optimizacion-de-ingresos

Este proyecto se divide en dos fases críticas para el crecimiento de una tienda online:

1. Estrategia: Priorización de hipótesis de marketing utilizando los frameworks ICE y RICE para maximizar el retorno de inversión (ROI).

2. Análisis de Datos: Evaluación de un Test A/B mediante el análisis de métricas acumuladas, conversión y significancia estadística para determinar el impacto real en los ingresos.
